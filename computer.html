About the Computer Science Games on This Site

These 12 free Computer Science games span three academic levels — from foundational digital literacy for primary school children through to university-level topics in algorithms, data structures and network protocols. Every game was created by Mwalimu Ronny, a practising Computer Science teacher based in Embu County, Kenya, drawing on firsthand experience of what students find challenging and what kinds of practice produce the best results.

Computer Science is a subject that rewards active engagement far more than passive reading. Understanding why a bubble sort makes the comparisons it does, or how binary search halves the search space with each step, requires tracing through the logic step by step — exactly what these games are designed to do. The Sorting Visualiser, Search Algorithms game and Recursion Tracer all build this procedural understanding in a way that static textbook diagrams cannot replicate.

Secondary CS — KCSE Computer Studies Alignment

The six secondary-level Computer Science games map directly to the Kenya Certificate of Secondary Education (KCSE) Computer Studies syllabus. Number System Converter covers binary, decimal, octal and hexadecimal conversions (Form 1–2), including Bin→Dec, Dec→Bin and Dec→Hex modes with immediate feedback on each attempt. Logic Gates covers the six fundamental gates — AND, OR, NOT, NAND, NOR and XOR — through truth table questions with an SVG gate diagram drawn for each question (Form 2–3). Programming Concepts covers pseudocode and Python tracing, including loops, conditionals, functions and variable assignment (Form 2–3). Data Representation covers bits, bytes, kilobytes, megabytes, image encoding (colour depth), sound sampling and lossless compression (Form 1–3). Sorting Visualiser provides animated, step-by-step visualisation of Bubble Sort, Selection Sort and Insertion Sort with comparison counts (Form 3). Cybersecurity Quiz covers phishing, malware, ransomware, firewalls, encryption, two-factor authentication and VPNs (Form 3–4).

University CS — Algorithms and Theory

The six university-level games tackle the theoretical foundations of Computer Science taught in first and second year undergraduate courses. Big-O Complexity presents code snippets in Python and asks students to identify the time complexity — covering O(1), O(n), O(n²), O(log n) and O(n log n). Data Structures tests knowledge of stacks (LIFO), queues (FIFO), binary trees, graphs, linked lists and hash tables — including operation names like push, pop, enqueue and dequeue. Boolean Algebra covers De Morgan's laws, identity laws, complement laws and expression simplification. Search Algorithms generates random sorted arrays and asks students to count how many comparisons Linear Search or Binary Search would need to find a given target — building genuine understanding of the difference in efficiency. Recursion Tracer presents Python recursive functions (factorial, Fibonacci, sum) and asks students to trace the output. Network Protocols covers the OSI model (7 layers), TCP vs UDP, HTTP/HTTPS ports, DNS, DHCP, FTP, SMTP and ARP.

Primary CS — Building Digital Literacy from the Ground Up

The six primary-level Computer Science games introduce the foundational ideas of computing in a format that is accessible to children from the age of 8, with no prior knowledge of computers or programming required. Binary Explorer uses a multiple-choice format to teach binary-to-decimal conversion, with a permanent reminder of the bit position values (128, 64, 32, 16, 8, 4, 2, 1) displayed beneath each question. ASCII Adventure teaches the ASCII character encoding system — that every letter, digit and symbol is stored as a number — through quiz questions about ASCII codes and their meanings. Algorithm Detective presents everyday algorithms (making tea, sending an email, brushing teeth) as shuffled step lists and asks students to tap pairs of steps to swap them into the correct order, building an intuition for sequential logic. Computer Parts Quiz covers the main hardware components — CPU, RAM, ROM, hard disk, GPU, motherboard, input devices and output devices — with 12 questions drawn from a randomised bank. Pattern Logic presents sequences of numbers, letters or emoji and asks students to identify the next item, building the abstract thinking that underpins programming loops and conditional logic. Internet Basics covers how the internet works — including HTTP, DNS, IP addresses, routers, Wi-Fi, browsers and web URLs — through 10 multiple-choice questions.

All games on this page are completely free, require no login and work on any device. Visit the About page to learn more about the site and its creator, or play the Mathematics games for the companion subject.
