Logic Gates and Truth Tables

A complete guide to the six fundamental logic gates — AND, OR, NOT, NAND, NOR and XOR — with full truth tables, real-world analogies and KCSE worked examples.

What Are Logic Gates?

A logic gate is an electronic circuit that takes one or more binary inputs (each either 0 or 1) and produces a single binary output (either 0 or 1), based on a specific logical rule. Logic gates are the fundamental building blocks of all digital circuits — from simple calculators to the processors inside your smartphone.

Every arithmetic operation your computer performs, every comparison it makes, and every decision in every program you run is ultimately carried out by combinations of millions of logic gates switching billions of times per second.

The behaviour of a logic gate is completely described by its truth table — a table that lists every possible combination of inputs and the output that results from each combination.

Remember: In logic gates, 1 means TRUE (or HIGH voltage) and 0 means FALSE (or LOW voltage). These correspond directly to the binary digits you learned in the binary number system.

The Six Fundamental Logic Gates

You need to know six logic gates for KCSE Computer Studies. Each gate has a name, a symbol, a Boolean expression and a truth table.

AND Gate

Output is 1 only when ALL inputs are 1

Boolean expression: Y = A · B (or A AND B)

Real-world analogy: A door that opens only when BOTH keys are inserted simultaneously.

How to Read a Logic Gate Question

KCSE questions typically give you a logic circuit diagram with gates connected together, provide specific input values, and ask you to determine the final output. Here is a systematic approach:

  1. Identify each gate in the circuit and its type (AND, OR, NOT, etc.).
  2. Start from the inputs on the left and work right toward the output.
  3. Calculate the output of each gate using its truth table.
  4. Feed that output as the input to the next gate.
  5. Continue until you reach the final output.

Worked Example 1 — Combined Gates

A circuit has inputs A=1, B=0, C=1. Gate 1 is AND (inputs A, B). Gate 2 is OR (inputs: output of Gate 1, and C). Find the output.

Step 1: Gate 1 (AND): A=1, B=0 → 1 AND 0 = 0

Step 2: Gate 2 (OR): output of Gate 1 = 0, C = 1 → 0 OR 1 = 1

Final output: 1

Worked Example 2 — Using De Morgan's Laws

Simplify: NOT(A AND B). According to De Morgan's first law, this equals NOT A OR NOT B.

If A=1, B=0: NOT(1 AND 0) = NOT(0) = 1

Verify with De Morgan: NOT A OR NOT B = NOT(1) OR NOT(0) = 0 OR 1 = 1 ✓

De Morgan's Laws

De Morgan's laws are two essential rules for simplifying Boolean expressions involving NOT, AND and OR:

  • First Law: NOT(A AND B) = NOT A OR NOT B. In other words, NAND = NOT A OR NOT B.
  • Second Law: NOT(A OR B) = NOT A AND NOT B. In other words, NOR = NOT A AND NOT B.

These laws are extremely useful for simplifying complex logic circuits and converting between gate types. They appear frequently in KCSE Paper 1 questions.

KCSE Tips for Logic Gates

  • Memorise the key outputs for each gate: AND outputs 1 only when all inputs are 1. OR outputs 0 only when all inputs are 0. NOT flips the input. NAND and NOR are the complements of AND and OR respectively. XOR outputs 1 only when inputs differ.
  • Draw truth tables carefully: Always list ALL possible input combinations in order (00, 01, 10, 11 for two inputs) so you do not miss any row.
  • Universal gates: Know that NAND and NOR are universal — you can be asked to draw a NOT gate using only NAND gates, for example.
  • Boolean notation: A·B means AND; A+B means OR; Ā means NOT A. Be comfortable with all three notations.

Summary

There are six fundamental logic gates: AND (output 1 only if all inputs are 1), OR (output 1 if any input is 1), NOT (inverts a single input), NAND (inverse of AND), NOR (inverse of OR), and XOR (output 1 only when inputs differ). Each gate is completely described by its truth table. NAND and NOR are universal gates. De Morgan's laws link NAND to NOT-OR and NOR to NOT-AND. Logic gates are the building blocks of all digital circuits and all computer processors.
