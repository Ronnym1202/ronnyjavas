About This Project

Ronny Edu Games is a free collection of interactive educational games covering Mathematics and Computer Science. Every game was designed and built by Mwalimu Ronny — a practising Mathematics and Computer Science teacher.

The site was created out of a simple belief: students learn better when they are engaged, challenged and having fun. Too many great learning tools are hidden behind subscriptions or app downloads. This site changes that.

Who Is It For?

The games are organised into three levels to cater for every stage of education:

  • Primary (Ages 8–13): Foundational number sense, basic operations, shapes, binary and computational thinking.
  • Secondary (Form 1–4): Aligned to the KCSE syllabus. Algebra, trigonometry, statistics, logic gates, programming and sorting algorithms.
  • University Level: Calculus, matrices, probability, data structures, complexity (Big-O) and Boolean algebra.

Our Principles

  • 🆓 Free forever — no subscriptions, no paywalls, no premium tiers.
  • 🚫 No advertisements — a distraction-free space built purely for learning.
  • 🔐 No login required — no accounts, no emails, no data collected.
  • 📱 Works everywhere — any device with a browser, including low-end phones.
  • 🎓 Curriculum-aligned — secondary games follow the KCSE Mathematics and Computer Studies syllabi.

Technology

The entire site is built with plain HTML, CSS and JavaScript — no frameworks, no dependencies, no trackers. It loads fast on any connection and works offline once cached.
